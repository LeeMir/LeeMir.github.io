##[Linux] 1. 리눅스 명령어

`ls` // 디렉토리의 내용을 출력한다
`setuid` `setgid` // root의 권한을 얻고싶을 때
`cd` // Change Directory(cd ~ : Home Directory)
`mkdir` // Make Directory
`rmdir` // Remove Directory(Directory가 비어있지 않으면 rmdir -r dir으로 삭제, -f : 물어보지 않고 삭제)
`cat` // 파일을 생성하거나 파일의 내용을 출력한다
