## 5강 - 포토샵 활용(이어서)

- - -

```
Ctrl + O : 파일 열기

Ctrl + K : Preference

Ctrl + J : 배경을 복사한 레이어 생성

V : 무브 / 아트보드

M : 사각 또는 원형 영역 선택 (Feather로 흐려짐 조절 가능), 영역 추가/제거/교차 선택 가능
ㄴ 영역 선택 후 Alt + Backspace / Ctrl + Backspace를 통해 전경색과 배경색을 채울 수 있음
ㄴ Shift를 누르고 드래그하면 항상 정사각형 또는 원을 그릴 수 있음
ㄴ Shift + Alt를 누르고 드래그하면 내가 누른 위치를 중심으로 그려짐
ㄴ Ctrl + D를 누르면 내가 선택했던 영역을 모두 지울 수 있음

L : Lasso - 내 마음대로 영역 선택 가능, Magnetic Lasso로 마우스 커서 따라 자동으로 영역 선택도 해줌

W : Quick Select(내가 클릭한 영역 자동 선택) / Magic Wand(사진에 있는 물건 자동 선택)
ㄴ 선택 후 Shift + Ctrl + I를 하면 반전 가능

C : Crop, 각도도 수정 가능

K : Frame - 원 또는 사각 영역에 다른 사진을 추가해 가져다 놓을 수 있음

I : Eyedropper - 스포이드, Ruler - 자, Note - 메모, Count - 카운트

J : Spot Healing Brush - 드래그해서 잡티 제거, Healing Brush - Alt + 클릭 후 그 영역을 기반으로 잡티 제거, Patch - 잡티를 제거할 영역을 그린 후 깨끗한 부분을 골라 붙임, Content Aware Move - 위치를 바꾸고 싶은 영역을 그려서 이동, Red Eye - 적안 제거
ㄴ Spot의 크기는 '['와 ']'로 조절할 수 있음

B : Brush

S : Clone Stamp - Alt + 클릭해서 찍은 영역을 다른 곳에 복사함

Y : History Brush - 열심히 갖고 놀았던 것을 지울 수 있음, Art History Brush - 이미지를 좀 더 예술적으로 만듦
```
